◆ VRoid SDKのインストール
ダウンロードしたVRoidSDK.unitypackageファイルを

Assets > Import Package > Custom Package...

からインポートすることでインストールが完了します

VRoid SDKはAssets/VRoidSDK以下にインストールされます。
Assets/VRoidSDKにREADME.mdがあるので、こちらもご一読ください。

-------------------------------------------------------------------------------------

◆ VRoid SDKのダウンロードと、アプリケーション登録を行ってください。

◇ アプリケーション登録
https://hub.vroid.com/oauth/applications
事前に登録いただいたVRoid Hubアカウントへログインした状態で以下URLへアクセスすると、OAuthアプリケーション登録が行えます。
ヘルプはこちら
https://vroid.pixiv.help/hc/ja/articles/900000277086-%E3%82%A2%E3%83%97%E3%83%AA%E3%82%B1%E3%83%BC%E3%82%B7%E3%83%A7%E3%83%B3%E8%A8%AD%E5%AE%9A%E3%81%AB%E3%81%A4%E3%81%84%E3%81%A6
